# Serverless-lead-Capture-on-AWS
It is a serverless eBook landing page application built using AWS services. It provides a static, responsive frontend hosted on Amazon S3 and a backend by API Gateway, AWS Lambda, and Amazon SES to handle secure form submissions and email delivery.A serverless architecture with proper handling of POST requests, CORS, and cloud-based email delivery.
